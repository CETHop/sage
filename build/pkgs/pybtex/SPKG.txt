= pybtex =

== Description ==

Pybtex is a drop-in replacement for BibTeX written in Python.

This package also installs pyparsing and PyYAML which are needed for pybtex.

== Special Update/Build Instructions ==

Make sure the easy_install commands in spkg-install work without internet
access.

We include a patch for pybtex to improve it's support for bibtex formats:

https://code.launchpad.net/~matthias-troffaes/pybtex/doctree/+merge/55170

The original version of this patch as of August 10, 2011 can be found in

patches/pybtex-doc_tree.patch.org

The file patches/pybtex-doc_tree.patch is rebased to apply to pybtex-0.15.

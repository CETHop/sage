= KASH =

== Description ==

This is a binary of KASH from:  http://www.math.tu-berlin.de/~kant/download.html

It works on OS X and Linux, but nothing else, since that's all that is available for KASH.
The Linux binary is 32-bit only, but will run on 64-bit boxes in 32-bit mode.
